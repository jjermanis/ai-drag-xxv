# ai-drag-xxv

Drag Race, Ho Ho Ho Style, using Chat GPT-5.

## Overview

Do you like Santa Claus? Do you like drag racing? If "yes" to both... then you'll be in for a treat.

This is a two-player race game. Two legendary gift givers race for the finish line. One player can also race, against a computer-led racer.


## Directions

### 2 Players

The main version of this game has two players. Here is how that game is played.

Each racer has their own button. The red player taps Q, and the green player taps P. At the center of the playing area are the racing lights. The light starts red. When the light turns green, the players can go.

Players only need to tap their button once to start moving. They will accelerate as fast as they can from there. Again: they only need to tap their button once.

If a player taps their button while the racing light is still red (before it is green), then that player loses.

The first Santa Claus to cross the finish line wins! If the two racers are very close, it can be a tie.

After each race, you can click Next Race at the bottom to start the next race. To delete the players won-loss-tie records, click on Clear Records.

### 1 Player

There is also support for an alternate version with just one player. To play this version, click on the 1 Player button in the lower-left corner.

The player controls the red player. The rules are the same: they tap Q to go, they will lose if they tap before the light turns green, they win if they cross the finish line first. In this mode, the computer controls the start of the green player.


## Mobile

This game has partial support on a mobile tablet. This game is unplayable on a smartphone.

When playing on a tablet, only 1 Player mode is supported. Be sure to tap the 1 Player button in the lower-left corner to start. To start each race, with your finger, tap on the red start button (toward the bottom, below the Swift Santa name). It is same as previously described: tap that button as soon as the racing light turns green. If you tap the button before the racing light turns green, you lose. The computer will control the green player.

To repeat, this game is unplayable on a smartphone.


## Details

Truly, there are no advantages to do anything other than clicking to start. Acceleration is purely random after starting - no tricks.
