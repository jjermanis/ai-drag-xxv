# ai-drag-xxv

Drag Race, Ho Ho Ho Style, using Chat GPT-5.

## Overview

Do you like Santa Claus? Do you like drag racing? If "yes" to both... then you'll be in for a treat.

This is a two-player race game. Two legendary gift givers race for the finish line. 


## Directions

Each racer has their own button. The red player taps Q, and the green player taps P. At the center of the playing area are the racing lights. When the light turns green, the players can go.

Players only need to tap the button once to start moving. They will accelerate as fast as they can from there. Again: they only need to tap the button once.

If a player taps their button while the racing light is still red (before it is green), then that player loses.

The first Santa Claus to cross the finish line wins! If the two racers are very close, it can be a tie.

After each race, you can click Reset Race at the bottom to have the next race. To delete the won-loss-tie records, click on Clear Records.


## Details

Truly, there are no advantages to do anything other than clicking to start. Acceleration is purely random after starting - no tricks.
